Pimiö

[pimiø] (Darkroom in Finnish) is a Javascript library to apply photo effects to the images on a website. It's currently under development but you can look at the source code of this page if you're interested.

Since it uses the canvas HTML element it only works on Firefox, Chrome, Safari and Opera. IE support should be possible by using FlashCanvas but it hasn't been tested (yet).
